# RustDesk Deploy Packager v0.1.0

## English

RustDesk Deploy Packager v0.1.0 is the first stable public release.

This release provides a bilingual Windows generator that creates a single RustDesk deployment EXE with embedded self-hosted server configuration. It supports downloading the latest official RustDesk Windows x64 installer, building a self-contained setup package, applying server settings automatically, enabling direct IP access, configuring startup, installing the service, and launching RustDesk after setup.

## 中文

RustDesk 私有化部署封装器 v0.1.0 是首个正式公开版本。

本版本提供一个中英文双语 Windows 生成器，可以生成内嵌私有化服务器配置的单文件 RustDesk 部署 EXE。支持下载官方最新版 RustDesk Windows x64 安装包、生成自包含安装包、自动写入服务器配置、开启 IP 直接访问、配置开机启动、安装服务，并在安装完成后自动打开 RustDesk。
