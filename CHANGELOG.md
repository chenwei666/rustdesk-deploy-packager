# Changelog / 更新日志

## v0.1.0 - 2026-06-05

### English

- Initial stable release.
- Added bilingual English/Simplified Chinese UI.
- Added modernized WinForms layout.
- Added custom application icon.
- Added latest official RustDesk Windows x64 installer download.
- Added one-file setup EXE generation.
- Generated installers apply self-hosted server configuration automatically.
- Generated installers enable direct IP access, configure startup, install service, and launch RustDesk.
- Removed all private default configuration values for public release.

### 中文

- 首个正式版本。
- 增加英文/简体中文双语界面。
- 重做 WinForms 界面布局。
- 增加自定义应用图标。
- 支持下载官方最新版 RustDesk Windows x64 安装包。
- 支持生成单文件安装 EXE。
- 生成的安装包会自动写入私有化服务器配置。
- 生成的安装包会自动开启 IP 直接访问、配置开机启动、安装服务并启动 RustDesk。
- 移除所有私有默认配置，适合公开发布。
